# Minimal Agent Context

## What this is
This is the minimal-agent example from the Based Agent framework.
It demonstrates all 7 core systems: loop, tools, permissions, memory, subagents, context, and prompts.

## Available tools
- file_read — read files in the current directory
- grep — search for patterns across files
- web_search — search the web (stub — configure a real provider)

## Notes for the agent
- You have access to a small tool set for demonstration purposes
- Prefer targeted reads over reading entire large files
- Report clearly when tasks are complete
